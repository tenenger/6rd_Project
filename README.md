## 4. 커밋 컨벤션

| commit   | 사용 예시         |

| feat     | 새로운 기능 추가  |
| refactor | 리팩토링          |
| fix      | 버그 수정         |
| Design   | CSS 스타일링      |
| docs     | 문서 수정         |
| remove   | 파일 삭제         |
| chore    | 패키지 매니저 수정 |
| deploy   | 배포              |
